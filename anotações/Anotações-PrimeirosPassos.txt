>> Como funcionam as tags <<

    <> O HTML é a estrutura de um site. então nele ira conter textos, titulos... </>
    <> tag - para declarar <tag> e para fechar </tag>
    <> tag titulo - <h1> ao <h6> por hierarquia </>
    <> para o paragrafo <p> </>
    <> Atraves dos links, podemos de fato fazer a navegação em diversas paginas </>
    <> As tags auxiliam também na hora de estilizar individualmente cada parte que eu demarcar </>
    <> tag <br> - tag de brack </>

>> Atributos << 

    <> São palavras utilizadas dentro das tags, normalmente nas de abertura, ou nas que fecham nela mesma, afim de alterar o comportamento dessa tag, ou até mesmo aplicar informações adicionais atraves de alguns atributos </>
    <> Atributo lang: atributo para declarar a linguagem da pagina </>
    <> Atributo abbr: serve para demarcar uma abreviação, como VIP, ou até mesmo HTML, que é uma abreviação para um conjunto de palavras, por exemplo </>

>> Entendendo as metatags <<

    <> tag meta, é uma tag que precisa de um atributo </>
    <> tags de configurações da pagina </>
    <> são inseridas dentro da tag head </>
    <> Uma tag que se fecha nela mesma </>
    <> Charset = Um padrão de escrita, o mais comum é o UTF-8 </>

>> Estrutura basica HTML <<

    <> O vscode possui um atalho para a estrutura basica do HTML </>
    <> Metatag viewport: pedindo ao navegador para seguir a escala sem zoom </>

>> Tags de titulo <<

    <> Por padrão, eles ja possuem uma fonte em negrito, tem um tamanho de fonte especifico, e tem um espaçamento entre eles. tanto inferior, quanto superior </>
    <> Não usar 2 titulos principais na mesma pagina, como recomendação </>
    <> Não escolho um titulo por conta da fonte pra isso temos o css </>

>> Outras tags <<

    <> Algumas tags servem para alterar como o texto fica </>
    <> <b> = tag para bold, negrito </>
    <> <i> = tag para italico </>

>> Listas <<

    <> para listas, temos 2 tipos. a ordenada, cujo a tag é "ol" (ordenery list), a lista não ordenada cujo a tag é ul(undeery list). E dentro de cada lista, temos a tag li (list itens) </>
    <> Por padrão, o html ja numera as listas para uma lista ordenada </>

>> Tabelas <<

    <> Tag para tabelas é "table". e dentro delas tem algumas tags para cada coisa </>
    <> para cada linha, a tag é "tr" </>
    <> e para cada coluna, tem a "td" </>
    <> Para criar deixar destacado de que uma linha da tabela é um indice, temos a tag "th", que ja vai tentar deixar em negrito, e tenta centralizar </>

>> Tags ancora (Links) <<

    <> Link é um movimento que nos leva a ir a outra pagina </>
    <> No html, a tag é o "a", que vem de ancora </>
    <> aqui no vs code, o atalho ja cria tb um atriburo "href", que é onde a gente poe a URL, referencia do caminho que queremos ancorar </>
    <> Ja por padrão, os links são coloridos com o Azul, e um tracejado embaixo. e se clicar, o link fica com uma cor azul um pouco mais escura, pra identificar que o link ja foi acessado </>

>> Atributos da tag ancora <<

    <> Target: alvo, aonde vai abrir esse link. temos as opções blank, parent, self e top. o mais ultilizado blank, que é para abrir numa aba vazia. o self serve para abrir na mesma aba, o que não é necessario apontar o atributo, logo que, de padrçao, o html ja usa ele como base </>
    <> rel: mais em relação a segurança da nossa pagina, ele permite ou não permite com que não autorize com que a pagina referenciada na tag, não possa acessar a nossa </>

>> Trabalhando com imagens <<

    <> A maneira mais comum de adicionar uma imagem. é usando a tag img </>
    <> Se fecha nela mesma </>
    <> atributos que acompanham ela é o SRC, que é onde passamos o caminho de onde esta a imagem, e o alt, que é onde podemos colocar um texto descritivo da imagem. Quanto mais descritivo, melhor </>
    <> Podemos não só usar imagens que esta no nosso computador, mas podemos usar imagens que estão online </>

>> Atributos adicionais da tag img <<

    <> Podemos, dentro da propria tag, declarar o tamanho de largura e altura da imagem </>
    <> width: atributo de largura, aqui podemos alterar o tamanho da imagem sem mudar a proporção dela </>
    <> height: atriburo de altura </>
