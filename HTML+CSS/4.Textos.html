Sendo a principal tag de texto, é usada para criar paragrafo

Mesmo tendo funcionalidade parecida com o uso dos parágrafos, os spans geralmente são utilizados para agrupar uma pequena informação

o b deixa o texto em negrito

o i deixa o texto em italico

o hr cria uma linha horizontal

o strong adiciona importância (SEO) no texto, diferente de bold, que só transforma em negrito
